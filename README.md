# 🛠️ C School & Personal Projects

Welcome to this repository! Here you'll find my projects, exercises, and workshops developed in **C** during my school years and personal study.

The repository contains both single-file exercises and more structured libraries. **Folder-based projects contain their own specific `README.md` file** with detailed instructions on how to compile and run them.
> **Note on Language:** The user interface, variable names, code comments and file names are written in Italian.

---
## 📂 Repository Structure

Here is how the repository is organized:

### 🎮 Single-File Programs / Games
These can be compiled directly using GCC:
* `📄 tris.c` - Tic-Tac-Toe game in C terminal (Input validation can be improved).
* `📄 impiccato.c` - Hangman game in C terminal (Current limitation: single words only; phrases cause input buffer issues, but it still works).
* `📄 calcolatrice_vettore_puntatori_a_funzioni.c` - A lightweight Command Line Interface (CLI) calculator that utilizes an array of function pointers to execute operations.

### 📦 Structured Projects
* `📂 libreria-persona/` - Didactic example of Encapsulation and Opaque Structs in standard C. Check its internal `README.md` for details!
* `📂 networking/` - A collection of network tools written in C.  Check its internal `README.md` for details!
---

## 🛠️ Global Prerequisites

To run the projects in this collection, make sure you have the following installed on your computer:

* **GCC (or Clang):** The standard C compiler (part of MinGW on Windows, or build-essential on Linux).
* **An IDE/Editor of your choice:** Dev-C++, VS Code, CLion or whatever you want.
* **Git:** To clone the repository.
