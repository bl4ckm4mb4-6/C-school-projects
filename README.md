# 🛠️ C School & Personal Projects

Welcome to this repository! Here you'll find my projects, exercises, and workshops developed in **C** during my school years and personal study.

The repository is organized into subfolders. **Each project contains its own specific `README.md` file** with detailed instructions on how to compile and run it.

## 🛠️ Global Prerequisites

To run the projects in this collection, make sure you have the following installed on your computer:

* **GCC (or Clang):** The standard C compiler (part of MinGW on Windows, or build-essential on Linux).
* **An IDE/Editor of your choice:** Dev-C++, VS Code, CLion or whatever you want.
* **Git:** To clone the repository.
