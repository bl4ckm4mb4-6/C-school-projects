# Networking projects

This projects implement networking in C.



## Projects:

* **pscan:** This program implements ports scanning thanks to socket.

* **lping:** This program is a ping tool that implements ICMP 



## 🚀 How to compile and run (GCC)
In Dev-C++, you compile the file first. Then in the command prompt or PowerShell, use this commands:

- For pscan

```bash
    C://your_path/pscan.exe <IP> <start_port> <end_port>
```

- For lping

```bash
    C://your_path/lping.exe <IP>
```

Both have help commands that are: `?`, `h`, `help`, `-h`, `--help` and `/?`.

Alternatively, you can insert the folder that contains pscan and lping in your PATH, so the command is simpler:

```bash
    pscan <IP> <start_port> <end_port>
```

```bash
    lping <IP>
```
